# SE Stock App

Modern inventeringsapp för flera lagerplatser (Malmö, Helsingborg, etc.).

## Funktioner
- Välj lagerplats
- Lägg till och uppdatera produkter
- Antal sparas automatiskt på telefonen
- Enkel och läsvänlig mobil UI
- Export av data (kommer snart)

## Kom igång

### Installation

1. Klona repot
2. Kör `flutter pub get`
3. Kör appen med:

   flutter run
